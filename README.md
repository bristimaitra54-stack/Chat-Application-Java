# Chat Application (Java)

## Features
- Multi-client chat system
- Real-time messaging
- Uses Java Sockets and Multithreading

## How to Run

1. Compile:
   javac Server.java
   javac Client.java

2. Run Server:
   java Server

3. Run Client:
   java Client

## Technologies Used
- Java
- Socket Programming
- Multithreading
